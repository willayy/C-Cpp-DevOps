# C-Cpp-DevOps
A base devops system for a c/cpp project including:
 * Issue templates
 * PR templates
 * Contribution guidelines
 * Branch protection rules for branch `prod`
 * License
 * GitHub Projects
 * GitHub Wiki
 * Github Discussions

 ### To build project
 ```shell
    cd C-Cpp-Devops/C-Cpp-proj
    mkdir build
    cd build
    cmake ..
    make # This will build the project executable and is optional
 ```
